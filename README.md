# B2B Sales Performance & Regional Analysis Dashboard

An end-to-end Excel analytics project built on a 200,000-row transactional sales dataset, simulating the kind of regional and product performance analysis a B2B sales organization would need.

## Dataset
[Product Sales Dataset (2023–2024)](https://www.kaggle.com/datasets/yashyennewar/product-sales-dataset-2023-2024) — Kaggle, 200,000 records.

## What's in this repo
- `product_sales_dataset_final.xlsx` — full working workbook (data, formulas, PivotTables, dashboard)
- `Kinlong_Sales_Project_Report.pdf` — detailed project write-up with methodology, findings, and recommendations

## Techniques Used
- Data cleaning (including resolving a mixed text/date-format column)
- SUMIFS / COUNTIFS, VLOOKUP / XLOOKUP
- PivotTables (revenue by region/month, top 10 products, customers by state)
- Conditional formatting for target-vs-actual KPI flagging
- Interactive dashboard with charts and slicers

## Key Findings
- East is the strongest region by both revenue and order volume; South underperforms on both.
- Electronics drives the largest share of revenue (40.4%) despite fewer orders than Clothing & Apparel.
- California and Arizona together account for a disproportionate share of total orders.

Full details, methodology, and recommendations are in the PDF report above.
